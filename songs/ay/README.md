# AY song examples

Representative AY-3-8910 / YM2149 examples for Atari ST, MSX, Amstrad CPC and Vectrex-style writing.

- Use `chip atari-st`, `chip msx`, `chip amstrad-cpc`, or `chip vectrex`.
- Noise percussion is often shared with bass on channel 3 via instrument swapping.
- Macro support examples include `vol_env`, `arp_env`, `pitch_env`, and `noise_rate_env`.
